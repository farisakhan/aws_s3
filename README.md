# aws_s3
# creating s3 bucket and put objects in bucket through local pc using aws cli

This project demonstrates how to create an S3 bucket and upload a file using AWS CLI.

Steps:
1. Configure AWS CLI using aws configure
2. Create bucket using aws s3 mb
3. Verify bucket using aws s3 ls
4. Upload file using aws s3 cp
5. Verify upload using aws s3 ls

Region used: us-west-2 

The file was uploaded from local system to AWS S3.
